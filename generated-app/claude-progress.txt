### Key Features
- **Dashboard**: Stats cards, burndown chart, velocity chart, pie chart, activity feed
- **Kanban Board**: @dnd-kit drag-and-drop, sprint filtering, issue navigation
- **Backlog**: Sprint sections, search/filter, create issue button
- **Sprints**: Lifecycle management (start/complete), progress bars
- **Epics**: Color-coded cards with progress tracking
- **Issue Detail**: Status/priority dropdowns, comments, metadata
- **Announcements**: CRUD, type filtering, pin/unpin
- **Settings**: Project config, seed data, danger zone
- **Keyboard Shortcuts**: g+d/b/l/s/e/p/a navigation, / for search
- **LocalStorage Persistence**: Full CRUD works without deployed backend
- **Mock Data Fallback**: Automatically switches to real API when VITE_API_URL set

- API Gateway: HTTP API with full CORS
- S3 + CloudFront for frontend hosting
- NOT YET DEPLOYED: No write AWS access in this environment

### To Deploy
1. Run `cd infrastructure && npx cdk deploy`
2. Get API URL from stack outputs
3. Set `VITE_API_URL=<api-url>` in frontend/.env
4. Call `POST /api/seed` to populate database
5. Frontend automatically uses real API
